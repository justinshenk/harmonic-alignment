# Harmonic Alignment Project

Cataloguing wisdom traditions as reference for multi-scale alignment—from individual development to institutional coordination to AGI values.

## Overview

35 contemplative, therapeutic, somatic, and philosophical traditions rated across 12 dimensions: mental health, cognitive, relational, and somatic effectiveness.

### GitHub Pages
Enable in Settings → Pages → Source: main branch

## Usage

- **Find Your Practice**: Interactive quiz to discover suitable contemplative practices based on your goals, time commitment, and circumstances
- **Compare View**: Sortable table showing all traditions with effectiveness ratings across dimensions
- **Evolution Tree**: Explore historical relationships between traditions
- **About**: Research methodology and citations
- **Character Development Model**: Practical framework synthesizing RL concepts with virtue ethics, featuring 15 gratitude endpoints for daily practice (with cognitive psychology citations)
